清空 Git 仓库全部历史记录
2023-03-19 18:06 label Git  
  1. 切换到 master 分支
git checkout master
  1. 创建一个干净的分支
git checkout --orphan new_master
  1. 提交全部文件
git add -A
git commit -m "Initial commit"
  1. 删除 master 分支
git branch -D master
  1. 将新分支重命名为 master
git branch -m master
  1. 强制推动到远程仓库
git push -f origin master
